# Portfolio Premium (React)

## What’s included
- React project with premium UI, dark/light mode toggle, and animations (Framer Motion).
- Components are modular; replace `YOUR_NAME`, projects and text with your details.

## How to run
1. Extract the ZIP.
2. `npm install` (or `yarn`) — installs dependencies including framer-motion.
3. `npm start`

## Files to edit
- `src/components/Config.js` — quick place to put your name, bio, projects, links and social.
- `src/App.css` — global styles (colors, fonts).
- Replace `YOUR_NAME` placeholders if any remain.
